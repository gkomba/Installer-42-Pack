# Instalação

## Para instalar o 42 Pack, basta seguir os passos abaixo:

## Baixe o repositorio
```bash
git clone https://github.com/gkomba/Installer-42-Pack.git
```

## Execute o instalador e siga as instruções na tela.
```bash
./installpack
```

## Para desisntalar:
```bash
./uninstallpack
```

## Para Actualizar
```bash
./updatepack
```

## Inicie o aplicativo e aproveite!
